# Kalinga Kshetra

**AI-Powered Heritage & Temple Guide Platform**
Status: In Progress, 2026

## About

Kalinga Kshetra is a web platform built to help people explore 500+ sacred temples across Odisha. It combines an interactive map, an AI-powered chatbot, and a pilgrimage planner to make discovering and visiting these heritage sites easier, with support for both Odia and English.

## Features

- Interactive map of temples across Odisha
- AI chatbot for answering visitor questions and planning trips
- Pilgrimage planner
- Civic complaint system for reporting heritage site issues
- Multilingual support (Odia and English)

## Getting Started

This is a front-end project with no build step required.

```bash
git clone https://github.com/Bidisha-26/kk.git
cd kk
```

Then open `kkh.html` in your browser to view the main page (`demo.html` is a separate demo/preview page).
